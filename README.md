# Selenium
## Selenium Web Automation using Jupyter Notebook
This project demonstrates basic web automation using Python and Selenium inside a Jupyter Notebook (SELENIUM.ipynb). It covers browser interaction, locating elements, and handling waits in a beginner-friendly way.
 ## Project Description
The notebook shows how to:
Launch a Chrome browser using Selenium
Open a website (example: W3Schools)
Locate elements using XPath
Use implicitly_wait()
Interact with web elements
This project is designed for beginners who want hands-on practice with Selenium in Jupyter Notebook.
## Technologies Used
Python 3.x
Selenium
Jupyter Notebook
Google Chrome
ChromeDriver
